## 문제

<https://www.acmicpc.net/problem/4375>

## 풀이

one 변수에는 11, 101, 1001, 10001... 과 같은 11과 관련된 숫자를 저장해, 계속 n과 나누어 떨어지는지, 아닌지 확인합니다.

```cpp
#include <iostream>
using namespace std;

int main() {
  int n;
  while (cin >> n) {
    int count = 1, one = 0;
    while (1) {
      one = one * 10 + 1;

      if (one % n == 0) break;
      one %= n;
      count++;
    }
    cout << count << "\n";
  }

  return 0;
}
```

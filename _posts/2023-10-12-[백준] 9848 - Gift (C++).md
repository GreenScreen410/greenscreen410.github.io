## 문제

<https://www.acmicpc.net/problem/9848>

## 풀이

이전 기록을 저장해 가며, 현재 값에서 뺐을 때 k 이상인지를 비교하면 됩니다.
저는 변수를 더 생성하고 싶지는 않아, 배열을 사용하였습니다.

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
  int n, k;
  cin >> n >> k;

  vector<int> v(n);
  for (int i = 0; i < n; i++) {
    cin >> v[i];
  }

  int count = 0;
  for (int i = 1; i < n; i++) {
    if (v[i - 1] - v[i] >= k) {
      count++;
    }
  }

  cout << count;

  return 0;
}
```

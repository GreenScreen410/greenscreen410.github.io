## 문제

<https://www.acmicpc.net/problem/14909>

## 풀이

단순한 짝수 판별 문제입니다. 단, EOF 처리가 필요합니다..

```cpp
#include <iostream>
using namespace std;

int main() {
  int N, count = 0;
  while (cin >> N) {
    if (N > 0) {
      count++;
    }
  }

  cout << count;

  return 0;
}
```

## 문제

<https://www.acmicpc.net/problem/1990>

## 풀이

팰린드롬인지 확인하는 함수 `palindrome`, 소수인지 확인하는 함수 `prime`을 만들고, 이 두개의 결과가 참일 때 해당 숫자를 출력하는 방법을 사용하였습니다.
그리고, $10,000,000$까지 돌되 b보다 커진다면 즉시 반복문을 탈출하여 시간 초과가 발생하지 않도록 하였습니다.

```cpp
#include <algorithm>
#include <cmath>
#include <iostream>
using namespace std;

bool palindrome(string s) {
  string s1 = s;
  string s2 = s1;
  reverse(s2.begin(), s2.end());

  if (s1 == s2) return true;
  else return false;
}

bool prime(int n) {
  if (n < 2) return false;

  for (int i = 2; i <= sqrt(n); i++) {
    if (n % i == 0) return false;
  }

  return true;
}

int main() {
  int a, b;
  cin >> a >> b;

  for (int i = a; i <= 10000000; i++) {
    if (i > b) break;

    if (palindrome(to_string(i)) && prime(i)) {
      cout << i << "\n";
    }
  }

  cout << "-1";

  return 0;
}
```

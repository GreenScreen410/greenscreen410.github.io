## 문제

<https://www.acmicpc.net/problem/15965>

## 풀이

에라토스테네스의 체 응용 문제입니다. sieve라는 배열을 만든 뒤, 이것이 true인 것만 v 배열에 넣어줍니다.

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
  int K;
  cin >> K;

  vector<bool> sieve(10000001, true);
  vector<int> v;
  for (int i = 2; i <= 10000000; i++) {
    if (sieve[i] == true) {
      v.push_back(i);
      for (int j = 2 * i; j <= 10000000; j += i) {
        sieve[j] = false;
      }
    }
  }

  cout << v[K - 1];

  return 0;
}

```

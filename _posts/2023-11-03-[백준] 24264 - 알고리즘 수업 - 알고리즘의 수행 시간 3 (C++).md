## 풀이

알고리즘 수업 1, 2를 완벽히 이해하였다면, 큰 문제 없이 풀 수 있을 것 같습니다.

```cpp
#include <iostream>
using namespace std;

int main() {
  long long n;
  cin >> n;

  cout << n * n << "\n2";

  return 0;
}
```

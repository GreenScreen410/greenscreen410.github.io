## 문제

https://www.acmicpc.net/problem/27251

## 풀이

전형적인 별 찍기 문제 같지만.. 100개 이상의 별을 출력해야 한다면 `...`을 출력해야 한다는 조건을 잘 봐야 합니다.

```cpp
#include <iostream>
using namespace std;

int main() {
  int n;
  cin >> n;

  for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= i * i; j++) {
      if (j > 100) {
        cout << "...";
        break;
      } else {
        cout << "*";
      }
    }
    cout << "\n";
  }

  return 0;
}
```

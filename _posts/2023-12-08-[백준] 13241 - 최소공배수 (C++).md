## 문제

<https://www.acmicpc.net/problem/13241>

## 풀이

유클리드 호제법 기초 문제입니다. 일부 언어에는 최소공배수를 구해주는 함수가 있으므로 그걸 사용해서 해결할 수도 있습니다.

```cpp
#include <iostream>
#include <numeric>
using namespace std;

int main() {
  long long A, B;
  cin >> A >> B;

  cout << lcm(A, B) << endl;

  return 0;
}
```

## 문제

<https://www.acmicpc.net/problem/10995>

## 풀이

딱히 특별할 것이 없는, 전형적인 별 찍기 문제입니다. 홀수 줄일때, 짝수 줄일때로 판별하였습니다.

```cpp
#include <iostream>
using namespace std;

int main() {
  int N;
  cin >> N;

  for (int i = 1; i <= N; i++) {
    if (i % 2 != 0) {
      for (int j = 0; j < N; j++) {
        cout << "* ";
      }
      cout << "\n";
    } else {
      for (int j = 0; j < N; j++) {
        cout << " *";
      }
      cout << "\n";
    }
  }

  return 0;
}
```
